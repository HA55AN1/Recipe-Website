# Project Setup Instructions

Thank you for reviewing my project. Please follow the steps below to run the website and database on your local machine using XAMPP.

---

## Requirements
- XAMPP must be installed (Download from https://www.apachefriends.org)
- Web browser (e.g. Chrome)

---

##  Included Files
- `Recipe Project.zip` — Contains all website files (PHP, HTML, CSS, JS)
- `recipe_db.sql` — SQL dump of the MySQL database

---

##  Setup Instructions

### Step 1: Extract the Website Files
1. Extract the `Recipe Project.zip` file.
2. Move the extracted folder (e.g. `Recipe Project`) into your: C:\xampp\htdocs\
3. Final path should look like: C:\xampp\htdocs\Recipe Project\

---

### Step 4: Start XAMPP
1. Open XAMPP Control Panel.
2. Click **Start** for:
- Apache
- MySQL

---

### Step 5: Import the Database
1. In your browser, go to: http://localhost/phpmyadmin
2. Click **Databases** tab > Create a new database  
Enter: `recipe_db` (must be same name as the `.sql` file)
3. Click the newly created database.
4. Click the **Import** tab.
5. Choose the `recipe_db.sql` file and click **Go**.

---

### Step 6: View the Website
Open the browser and go to: http://localhost/Recipe Project


---

You're ready to use the website locally.
